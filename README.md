# ⛽ Análise Exploratória do Preço da Gasolina no Brasil

## 📌 Sobre o Projeto

Este projeto tem como objetivo realizar uma análise exploratória dos dados de preço da gasolina no Brasil, buscando identificar padrões, variações regionais e possíveis fatores associados às diferenças de valores.

O projeto foi desenvolvido como parte do módulo da EBAC, com foco em aplicação prática de:

- Python
- Manipulação de dados
- Análise exploratória (EDA)
- Versionamento com Git
- Publicação no GitHub

---

## 🎯 Objetivos da Análise

- Analisar a variação do preço da gasolina entre municípios
- Identificar padrões regionais
- Explorar possíveis relações com indicadores econômicos
- Praticar boas práticas de versionamento com Git
